# TODO Group (todo-group)
The TODO Group is an open community under the Linux Foundation of practitioners collaborating on best practices, tools, and guidance for running open source program offices (OSPOs). It provides guides, case studies, and the OSPO landscape for organizations managing their open source strategies.
